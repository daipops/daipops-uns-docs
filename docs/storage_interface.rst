    1. 登录校验接口
    ======================================
    1.1. 接口功能
    --------------------------------------
    登录校验, 验证dmid的合法性
    
    1.2. 请求方式
    --------------------------------------
    POST /api/v1/validate
    
    1.3. 请求参数说明
    --------------------------------------
    1.3.1. 请求头部(HTTP Header)
    
    +--------------+--------------+--------------+----------------------------------------+
    |    字段名    |   是否必选   |   数据类型   | 描述                                   |
    +==============+==============+==============+========================================+
    | dmid         | 是           | String       | 联系DM技术支持获得                     |
    +--------------+--------------+--------------+----------------------------------------+
    
    1.3.2. 请求体
    
    +-----------------+---------------+---------------+-----------------------------------------------------------------------------------------+
    | 字段名          | 是否必选      | 数据类型      | 描述                                                                                    |
    +-----------------+---------------+---------------+-----------------------------------------------------------------------------------------+
    
    1.4. 正确请求示例
    --------------------------------------
    .. code-block::
    
        Header:
        {
            "dmid": "xxx"
        }
    
    1.5. 返回数据说明(Content-type: JSON)
    ------------------------------------------------------------------
    
    +---------------+---------------------+---------------+--------------------------------------------------------+
    | 字段名        | 是否必选            | 数据类型      | 说明                                                   |
    +===============+=====================+===============+========================================================+
    | code          | 是                  | Int           | code为0时, 业务正常, code不为0时, 业务出错             |
    +---------------+---------------------+---------------+--------------------------------------------------------+
    | data          | 是                  | JSONObject    |   空                                                   |
    +---------------+---------------------+---------------+--------------------------------------------------------+
    | msg           | 是                  | String        | 业务运行辅助定位信息                                   |
    +---------------+---------------------+---------------+--------------------------------------------------------+
    
    data字段说明
    
    +---------------+---------------+--------------------------------------------------------+
    | 字段名        | 数据类型      | 说明                                                   |
    +---------------+---------------+--------------------------------------------------------+
    
    1.6. 正确返回示例
    --------------------------------------
    .. code-block::
    
        {
            "code": 0,
            "data": {},
            "msg": "OK"
        }
    
    1.7. 错误返回示例
    --------------------------------------
    .. code-block::

    3. 任务推送接口
    ======================================
    3.1. 接口功能
    --------------------------------------
    推送任务对应的文件列表，这些文件列表表示即将上传的文件名，包含路径。
    
    3.2. 请求方式
    --------------------------------------
    POST /api/v1/task/upload
    
    3.3. 请求参数说明
    --------------------------------------
    3.3.1. 请求头部(HTTP Header)
    
    +--------------+--------------+--------------+----------------------------------------+
    |    字段名    |   是否必选   |   数据类型   | 描述                                   |
    +==============+==============+==============+========================================+
    | dmid         | 是           | String       | 联系DM技术支持获得                     |
    +--------------+--------------+--------------+----------------------------------------+
    
    3.3.2. 请求体
    
    +-----------------+---------------+---------------+-----------------------------------------------------------------------------------------+
    | 字段名          | 是否必选      | 数据类型      | 描述                                                                                    |
    +-----------------+---------------+---------------+-----------------------------------------------------------------------------------------+
    | taskid          | 是            | String        | 任务唯一码                                                                              |
    +-----------------+---------------+---------------+-----------------------------------------------------------------------------------------+
    | file_names      | 是            | String        | 文件列表                                                                                |
    +-----------------+---------------+---------------+-----------------------------------------------------------------------------------------+
    3.4. 正确请求示例
    --------------------------------------
    .. code-block::
    
        Header:
        {
            "dmid": "xxx",
        }
        
        Body:
        {
            "taskid":"93b2dcbe-eb9f-11eb-a269-b42e996650e9",
            "file_names":"/xxx/aaa1.wav,/xxx/vvv2.zip,/xxx/vvvv3.zip"
        }
    
    3.5. 返回数据说明(Content-type: JSON)
    ------------------------------------------------------------------
    
    +---------------+---------------------+---------------+--------------------------------------------------------+
    | 字段名        | 是否必选            | 数据类型      | 说明                                                   |
    +===============+=====================+===============+========================================================+
    | code          | 是                  | Int           | code为0时, 业务正常, code不为0时, 业务出错             |
    +---------------+---------------------+---------------+--------------------------------------------------------+
    | data          | 是                  | JSONObject    | 具体的返回数据(见data字段说明)                         |
    +---------------+---------------------+---------------+--------------------------------------------------------+
    | msg           | 是                  | String        | 业务运行辅助定位信息                                   |
    +---------------+---------------------+---------------+--------------------------------------------------------+
    
    data字段说明
    
    +---------------+-------------------+--------------------------------------------------------+
    | 字段名        | 数据类型          | 说明                                                   |
    +===============+===================+========================================================+
    |  XXXX         | "String":"String" | 文件名及其唯一码_id                                    |
    +---------------+-------------------+--------------------------------------------------------+
    
    3.6. 正确返回示例
    --------------------------------------
    .. code-block::
    
        {
            "code": 0,
            "data":  {
                "/aaa1.wav": "60fe31466bbaa37fb2847a81",
                "/bb/vvvv3.zip": "60fe31466bbaa37fb2847a83",
                "/test/vvv2.zip": "60fe31466bbaa37fb2847a82"
            },
            "msg": "OK"
        }
    
    3.7. 错误返回示例

    4. 文件上传接口
    ======================================
    4.1. 接口功能
    --------------------------------------
    上传文件, 通知DMAI工作人员进行特征分析
    
    4.2. 请求方式
    --------------------------------------
    POST /api/v1/storage/upload
    
    4.3. 请求参数说明
    --------------------------------------
    4.3.1. 请求头部(HTTP Header)
    
    +--------------+--------------+--------------+----------------------------------------+
    |    字段名    |   是否必选   |   数据类型   | 描述                                   |
    +==============+==============+==============+========================================+
    | dmid         | 是           | String       | 联系DM技术支持获得                     |
    +--------------+--------------+--------------+----------------------------------------+
    
    4.3.2. 请求体(multipart/form-data)
    
    +-----------------+---------------+---------------+-----------------------------------------------------------------------------------------+
    | 字段名          | 是否必选      | 数据类型      | 描述                                                                                    |
    +=================+===============+===============+=========================================================================================+
    | file            | 是            | file          | 待上传文件                                                                              |
    +-----------------+---------------+---------------+-----------------------------------------------------------------------------------------+
    | path            | 是            | String        | 上传文件的完整路径名，统一用'/'分隔路径,以'/'开头。                                     |
    +-----------------+---------------+---------------+-----------------------------------------------------------------------------------------+
    | flag            | 是            | String        | 本任务的最后一个文件则设置为'1'，否则设置为'0'                                          |
    +-----------------+---------------+---------------+-----------------------------------------------------------------------------------------+
    | taskid          | 是            | String        | 任务唯一码                                                                              |
    +-----------------+---------------+---------------+-----------------------------------------------------------------------------------------+
    | _id             | 是            | String        | 文件或文件夹唯一码                                                                      |
    +-----------------+---------------+---------------+-----------------------------------------------------------------------------------------+
    
    4.4. 正确请求示例
    --------------------------------------
    .. code-block::
    
        Header:
        {
            "dmid": "xxx",
            "Content-Type": "multipart/form-data"
        }
        -----------------------------------------------------------------------
        Body:
        {
            "file": xxx.zip,
            "path": "/test,
            "flag": "0",
            "taskid": "60fa79228545cb9280e2cc42",
            "_id": "60fe1e416622de0a700d5d45"
        }
    
    4.5. 返回数据说明(Content-type: JSON)
    ------------------------------------------------------------------
    
    +---------------+---------------------+---------------+--------------------------------------------------------+
    | 字段名        | 是否必选            | 数据类型      | 说明                                                   |
    +===============+=====================+===============+========================================================+
    | code          | 是                  | Int           | code为0时, 业务正常, code不为0时, 业务出错             |
    +---------------+---------------------+---------------+--------------------------------------------------------+
    | data          | 是                  | JSONObject    | 具体的返回数据(见data字段说明)                         |
    +---------------+---------------------+---------------+--------------------------------------------------------+
    | msg           | 是                  | String        | 业务运行辅助定位信息                                   |
    +---------------+---------------------+---------------+--------------------------------------------------------+
    
    data字段说明
    
    +---------------+---------------+--------------------------------------------------------+
    | 字段名        | 数据类型      | 说明                                                   |
    +===============+===============+========================================================+
    | url           | String        | 已上传文件的下载地址                                   |
    +---------------+---------------+--------------------------------------------------------+
    
    4.6. 正确返回示例
    --------------------------------------
    .. code-block::
    
        {
            "code": 0,
            "data": {
                "url": "http://xxx/test/xxx.zip"
            },
            "msg": "OK"
        }
    
    4.7. 错误返回示例
    --------------------------------------
    .. code-block::
    
        {
            "code": 401,
            "data": {},
            "msg": "Unauthorized"
        }
    
    
        {
            "code": 1001,
            "data": {},
            "msg": "Upload file fail"
        }
    
    4.8. 错误码
    --------------------------------------
    +----------+------------------------------------------------+------------------------+
    | 错误码   | 说明                                           | 修复方法               |
    +==========+================================================+========================+
    | 401      | 鉴权失败                                       | 联系DM技术支持获取dmid |
    +----------+------------------------------------------------+------------------------+

    7. 文件列表接口
    ======================================
    7.1. 接口功能
    --------------------------------------
    显示某个任务对应的文件列表
    
    7.2. 请求方式
    --------------------------------------
    GET /api/v1/task/list?c_page={}&p_size={}
    
    7.3. 请求参数说明
    --------------------------------------
    7.3.1. 请求头部(HTTP Header)
    
    +--------------+--------------+--------------+------------------------------------------------------------------------+
    |    字段名    |   是否必选   |   数据类型   | 描述                                                                   |
    +==============+==============+==============+========================================================================+
    | dmid         | 是           | String       | 联系DM技术支持获得                                                     |
    +--------------+--------------+--------------+------------------------------------------------------------------------+
    | taskid       | 否           | String       |任务id,若存在则获取该任务下的全部文件，否则获取dmid的全部已上传的文件   |
    +--------------+--------------+--------------+------------------------------------------------------------------------+
    
    7.3.2. 查询参数
    
    +--------------+--------------+--------------+------------------------------------------------------------------------+
    |    字段名    |   是否必选   |   数据类型   | 描述                                                                   |
    +==============+==============+==============+========================================================================+
    | p_size       | 是           | Int          | 单个页面记录最大条数                                                   |
    +--------------+--------------+--------------+------------------------------------------------------------------------+
    | c_page       | 是           | Int          | 当前页                                                                 |
    +--------------+--------------+--------------+------------------------------------------------------------------------+
    
    7.4. 正确请求示例
    --------------------------------------
    .. code-block::
    
        GET /api/v1/task/list?c_page=1&p_size=10
    
    7.5. 返回数据说明(Content-type: JSON)
    ------------------------------------------------------------------
    
    +---------------+---------------------+---------------+--------------------------------------------------------+
    | 字段名        | 是否必选            | 数据类型      | 说明                                                   |
    +===============+=====================+===============+========================================================+
    | code          | 是                  | Int           | code为0时, 业务正常, code不为0时, 业务出错             |
    +---------------+---------------------+---------------+--------------------------------------------------------+
    | data          | 是                  | JSONObject[]  | 具体的返回数据(见data字段说明)                         |
    +---------------+---------------------+---------------+--------------------------------------------------------+
    | msg           | 是                  | String        | 业务运行辅助定位信息                                   |
    +---------------+---------------------+---------------+--------------------------------------------------------+
    
    data字段说明
    
    +-----------------------------+----------------+--------------------------------------------------------+
    | 字段名                      | 数据类型       | 说明                                                   |
    +=============================+================+========================================================+
    | filename                    | String         | 文件或文件夹名称                                       |
    +-----------------------------+----------------+--------------------------------------------------------+
    | _id                         | String         | 文件或文件夹唯一码                                     |
    +-----------------------------+----------------+--------------------------------------------------------+
    | file_size                   | String         | 文件大小，单位KB                                       |
    +-----------------------------+----------------+--------------------------------------------------------+
    | upload_time                 | String         | 上传时间                                               |
    +-----------------------------+----------------+--------------------------------------------------------+
    | total_record                | Int            | 总记录数                                               |
    +-----------------------------+----------------+--------------------------------------------------------+

    +---------------+---------------------+---------------+--------------------------------------------------------+
    
    data字段说明
    
    +-----------------------------+----------------+--------------------------------------------------------+
    | 字段名                      | 数据类型       | 说明                                                   |
    +=============================+================+========================================================+
    | max_single_round_files      | Int            | 单次最多可选择文件数目,  默认6000个文件                |
    +-----------------------------+----------------+--------------------------------------------------------+
    | max_single_round_files_size | Int            | 单次最大上传大小,单位GB, 默认5GB                       |
    +-----------------------------+----------------+--------------------------------------------------------+
    | single_dmid_size            | Int            | 单个dmid最大占用空间,单位GB, 默认50GB                  |
    +-----------------------------+----------------+--------------------------------------------------------+
    | auto_clean_days             |  Int           | 数据上传auto_clean_days天后自动清理,初始值30           |
    +-----------------------------+----------------+--------------------------------------------------------+
    
    8.6. 正确返回示例
    --------------------------------------
    .. code-block::
    
        {
            "code": 0,
            "data": {
                     "max_single_round_files":6000,
                     "max_single_round_files_size":5,
                     "single_dmid_size":50,
                     "auto_clean_days":30
                    },
            "msg": "OK"
        }
    
    8.7. 错误返回示例
    --------------------------------------
    .. code-block::
    
       {
            "code": 401,
            "data": {},
            "msg": "Unauthorized"
       }

    | 401      | 鉴权失败                                       | 联系DM技术支持获取dmid |
