概述
======================================

DMAI非结构化数据特征提取, 针对双通道大于8kHz采样率语音, 做离线特征提取。


使用限制
======================================


1.支持双通道的.wav、.mp3格式的录音文件特征提取，且需要申明哪个通道是客服语音。

2.支持采样率大于等于8k的语音。

3.为保证结果的有效性，音频时长小于1秒，或语音文件中无有效对话，将不进行分析。

4.音频时长需小于1小时。
